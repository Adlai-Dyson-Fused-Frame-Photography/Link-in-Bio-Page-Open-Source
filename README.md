# A Free to use Link in Bio Page
#### Created by Adlai Dyson | Fused Frame Photography

Created using HTML and Tailwind CSS
See example [here](https://adffp.uk/linktree-example)

### Features
- All elements are easily editable (comments are left in the file to ease editing)
- Animated background pulse
- Glow when hovering over certain elements
- Slight Glassmorphism
- Optimised for both desktop and mobile devices; fast loading.
- Includes a profile picture and call to action banner.

### How to edit & deploy?
The best way to start using this template is to:
- Fork this repo,
- Edit the **index.html** file locally (in an app of your choice, Notepad++,. CotEditor, VS Code...),
- Commit and push,
- Then publish to a platform of your choice (**Cloudflare Pages or Github Pages recommended**)

**or**

- You can download the index.html file from this repo and,
- Edit it in an app of your choice, Notepad++,. CotEditor, VS Code...,
- Then upload/publish/host on a different platform of your choice

### Customisation
As the file is pure HTML you can customise anything that you want.

For those who don't know there way around HTML too much, ive included little comments above certain elements to highligh what they are and what to change. (for example; **<!-- Change Website Title Here -->**).

The icons used are from a FontAwesome Kit. You will need to update this to your specific kit (free account) to continue using.
